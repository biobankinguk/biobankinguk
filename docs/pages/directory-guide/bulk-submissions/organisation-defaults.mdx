# Organisation Defaults

Use of the Bulk Submissions facility requires that some default Collection details are set for your Organisation.

All Collections created from data sent through the Bulk Submissions service will adopt the details configured here.

## Setting the Defaults

<DifferentThemesAlert />

1. Log in to your account
   <ImageBox alt="Log in Menu Item" src="/i/menu/menu_login.png" />
1. Go to your Organisation's management area
   <ImageBox
     alt="Organisation Management Menu Item"
     src="/i/menu/menu_manage-org.png"
   />
1. In the **Submissions** section, select the desired defaults
   <ImageBox
     alt="Submissions Organisation Defaults"
     src="/i/submissions/org-defaults.png"
   />
